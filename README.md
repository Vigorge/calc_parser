# Разбор арифметических выражений 
Реализация простого разбора скобочных арифметических выражений путём синтаксического и лексического анализа.
### Грамматика:
	<E>  ::= <T> <E’>. 
	<E’> ::= + <T> <E’> | - <T> <E’> | . 
	<T>  ::= <F> <T’>. 
	<T’> ::= * <F> <T’> | / <F> <T’> | . 
	<F>  ::= <number> | <var> | ( <E> ) | - <F>.
